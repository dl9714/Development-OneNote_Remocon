# 전자필기장 복제

## 주문번호

SK-002

## Trigger

사용자가 OneNote 전자필기장을 복사하거나 같은 구조로 복제해 달라고 요청할 때

## Instructions

목표:
- 사용자가 지정한 원본 OneNote 전자필기장을 새 전자필기장으로 복제한다.
- 복제 대상 이름은 항상 `코덱스-{원본 전자필기장명}` 형식으로 정한다.

이름 기준:
- 원본이 `사업체-주식 바이럴`이면 대상은 `코덱스-사업체-주식 바이럴`이다.
- 사용자가 대상 이름을 따로 적어도 `코덱스-` 접두사가 없으면 `코덱스-`를 붙인다.
- 사용자가 명시적으로 다른 이름 규칙을 지시한 경우에만 그 지시를 우선한다.

복제 기준:
- 원본의 섹션 그룹, 섹션, 페이지 제목, 페이지 내용을 유지한다.
- OneNote 내부 휴지통이나 삭제 보관 영역은 일반 작업 대상에서 제외한다.
- 실제 OneNote 조작 방식은 코덱스 전용 OneNote 지침을 따른다.

검증:
- 원본과 대상의 활성 섹션 그룹 수, 섹션 수, 페이지 수가 일치하는지 확인한다.
- 필요하면 대표 페이지의 제목과 본문 일부가 대상에도 들어갔는지 확인한다.

보고:
- 원본 전자필기장 이름, 대상 전자필기장 이름, 복제한 섹션 그룹/섹션/페이지 수, 검증 결과만 짧게 남긴다.
