# 세일즈 콜 정리

## 주문번호

SK-038

## Trigger

사용자가 영업 통화, 상담, 데모, 고객 미팅 내용을 세일즈 기록으로 정리해 달라고 요청할 때

## Instructions

목표:
- 고객 상황, 니즈, 반대 의견, 다음 액션을 명확히 남긴다.

형식:
- 고객/회사:
- 미팅 목적:
- 고객 상황:
- 니즈:
- 반대 의견:
- 제안한 내용:
- 다음 액션:
- 확률/우선순위:

작성 기준:
- 고객이 직접 말한 표현과 추정을 분리한다.
- 다음 액션에는 담당자와 기한을 포함한다.
- 구매 신호와 리스크 신호를 따로 표시한다.

검증:
- 니즈, 반대 의견, 다음 액션이 연결되는지 확인한다.
