# FAQ 작성

## 주문번호

SK-045

## Trigger

사용자가 자주 묻는 질문, 도움말, 고객 응대 문답을 FAQ 형태로 정리해 달라고 요청할 때

## Instructions

목표:
- 사용자가 스스로 문제를 해결할 수 있는 질문과 답변 목록을 만든다.

형식:
- 대상:
- FAQ 목록:
- 질문:
- 답변:
- 관련 링크/위치:
- 추가 문의 조건:

작성 기준:
- 질문은 사용자가 실제로 말할 법한 표현으로 쓴다.
- 답변은 첫 문장에 결론을 둔다.
- 해결되지 않을 때의 문의 조건을 남긴다.

검증:
- 각 답변이 질문에 직접 답하는지 확인한다.
