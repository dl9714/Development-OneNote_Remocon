# 나의 기본 글쓰기 형식

## 주문번호

SK-001

## Trigger

OneNote에 새 페이지를 만들거나 Codex가 사용자의 메모/업무 기록을 작성할 때 사용한다.

## Instructions

목표:
- 사용자의 기본 글쓰기 형식을 OneNote 페이지에 일관되게 적용한다.

형식:
- 제목:
- 목적:
- 핵심 내용:
- 다음 행동:

검증:
- 생성된 페이지 제목과 본문을 GetPageContent로 확인한다.

OneNote 작성 기준:

- 화면 클릭보다 OneNote COM API를 우선 사용한다.
- 대상 섹션 ID가 있으면 `CreateNewPage + GetPageContent + UpdatePageContent`로 작성한다.
- 생성 후 `GetPageContent`로 제목과 본문이 들어갔는지 확인한다.
- 가능하면 마지막에 `NavigateTo(pageId)`로 새 페이지를 화면에 띄운다.

## OneNote COM Guardrails

- 화면 클릭보다 OneNote COM API를 우선 사용한다.
- 대상 ID가 있으면 `GetHierarchy` 전체 탐색보다 ID 직접 호출을 우선한다.
- 검증은 `GetHierarchy` 또는 `GetPageContent`를 우선 사용한다.
