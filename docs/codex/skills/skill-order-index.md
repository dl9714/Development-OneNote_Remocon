# 사용자 스킬 주문번호표

사용자가 주문번호로 스킬을 지시하면 이 표에서 해당 Markdown 파일을 찾아 따른다. OneNote 조작 방식은 `docs/codex/instructions`의 코덱스 전용 지침에서 관리한다.

| 주문번호 | 스킬 이름 | 파일 | 호출 조건 |
| --- | --- | --- | --- |
| SK-001 | 새 코덱스 스킬 복사본 | `새-코덱스-스킬-복사본.md` | 이 스킬을 쓸 상황을 적는다 |
| SK-002 | 전자필기장 복제 | `전자필기장-복제.md` | 사용자가 OneNote 전자필기장을 복사하거나 같은 구조로 복제해 달라고 요청할 때 |
| SK-015 | 나의 기본 글쓰기 형식 | `나의-기본-글쓰기-형식.md` | 사용자가 정리된 글쓰기 형식으로 OneNote 페이지 작성을 요청할 때 |
