# 의사결정 기록

## 주문번호

SK-017

## Trigger

사용자가 여러 선택지 중 하나를 고르거나 결정 배경을 남겨 달라고 요청할 때

## Instructions

목표:
- 결정 사항과 근거를 나중에 다시 봐도 이해되게 기록한다.

형식:
- 결정 제목:
- 결정 내용:
- 배경:
- 선택지:
- 판단 기준:
- 선택한 이유:
- 리스크:
- 후속 작업:

작성 기준:
- 선택하지 않은 대안도 짧게 남긴다.
- 결정 근거와 개인 선호를 구분한다.
- 되돌릴 수 있는 결정인지, 되돌리기 어려운 결정인지 표시한다.

검증:
- 결정 내용, 이유, 리스크, 후속 작업이 모두 있는지 확인한다.
