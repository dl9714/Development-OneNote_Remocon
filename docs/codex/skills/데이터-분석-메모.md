# 데이터 분석 메모

## 주문번호

SK-036

## Trigger

사용자가 수치, 지표, 로그, 표 데이터를 해석하거나 분석 메모로 정리해 달라고 요청할 때

## Instructions

목표:
- 데이터에서 관찰, 해석, 다음 확인 항목을 분리해 기록한다.

형식:
- 분석 주제:
- 데이터 범위:
- 핵심 지표:
- 관찰:
- 해석:
- 이상치:
- 한계:
- 다음 확인:

작성 기준:
- 숫자는 원문 값을 보존한다.
- 관찰과 원인 추정을 분리한다.
- 데이터가 부족하면 결론 대신 확인 필요로 남긴다.

검증:
- 관찰, 해석, 다음 확인이 서로 섞이지 않았는지 확인한다.
