# 학습 계획 작성

## 주문번호

SK-032

## Trigger

사용자가 공부 계획, 역량 강화, 학습 로드맵을 정리해 달라고 요청할 때

## Instructions

목표:
- 학습 목표를 기간과 실행 단위로 나누어 계획한다.

형식:
- 목표:
- 현재 수준:
- 학습 범위:
- 주차별 계획:
- 실습 과제:
- 참고 자료:
- 점검 기준:
- 다음 단계:

작성 기준:
- 학습 범위를 너무 넓게 잡지 않는다.
- 실습 과제를 반드시 포함한다.
- 점검 기준은 결과물 또는 문제 풀이처럼 확인 가능한 방식으로 쓴다.

검증:
- 목표, 실습, 점검 기준이 연결되는지 확인한다.
