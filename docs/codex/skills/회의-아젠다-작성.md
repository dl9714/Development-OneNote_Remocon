# 회의 아젠다 작성

## 주문번호

SK-050

## Trigger

사용자가 회의 전에 논의 안건, 목표, 준비물, 결정해야 할 항목을 정리해 달라고 요청할 때 사용한다.

## Instructions

목표:
- 회의가 잡담으로 흐르지 않도록 목적, 안건, 결정 항목을 먼저 정리한다.

형식:
- 회의명:
- 회의 목적:
- 참석자:
- 사전 준비:
- 안건:
- 결정해야 할 것:
- 공유만 할 것:
- 예상 쟁점:
- 시간 배분:
- 회의 후 산출물:

작성 기준:
- 안건은 우선순위 순서로 정렬한다.
- 결정이 필요한 항목과 단순 공유 항목을 분리한다.
- 준비물이 없으면 "없음"보다 "확인 필요"를 우선 사용한다.

검증:
- 회의 목적, 결정 항목, 산출물이 명확한지 확인한다.
