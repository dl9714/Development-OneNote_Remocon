# 실험 검증 기록

## 주문번호

SK-022

## Trigger

사용자가 테스트, 실험, A/B, 검증 결과를 기록하거나 비교해 달라고 요청할 때

## Instructions

목표:
- 실험 목적, 조건, 결과, 판단을 재현 가능하게 기록한다.

형식:
- 실험명:
- 가설:
- 조건:
- 절차:
- 결과:
- 판단:
- 한계:
- 다음 실험:

작성 기준:
- 결과와 판단을 분리한다.
- 표본, 조건, 실행 시간을 가능하면 남긴다.
- 실패한 실험도 배운 점을 기록한다.

검증:
- 가설과 결과가 같은 기준으로 비교되는지 확인한다.
