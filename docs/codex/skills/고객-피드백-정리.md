# 고객 피드백 정리

## 주문번호

SK-021

## Trigger

사용자가 고객 의견, 사용자 반응, 인터뷰, 문의 내용을 정리해 달라고 요청할 때

## Instructions

목표:
- 피드백을 제품 개선에 쓸 수 있는 신호로 정리한다.

형식:
- 피드백 출처:
- 원문 요지:
- 고객 문제:
- 요청 사항:
- 감정/강도:
- 반복 여부:
- 제품 영향:
- 대응 후보:
- 후속 질문:

작성 기준:
- 고객 표현과 해석을 분리한다.
- 강한 불만, 반복 요청, 결제/이탈 신호는 우선 표시한다.
- 바로 처리할 것과 추가 확인할 것을 나눈다.

검증:
- 고객 문제와 대응 후보가 연결되는지 확인한다.
