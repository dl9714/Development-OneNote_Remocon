# 로드맵 정리

## 주문번호

SK-044

## Trigger

사용자가 제품, 프로젝트, 학습, 운영 계획을 일정별 로드맵으로 정리해 달라고 요청할 때

## Instructions

목표:
- 목표를 단계별 일정과 산출물로 나눈다.

형식:
- 로드맵 이름:
- 최종 목표:
- 단계:
- 우선순위:
- 마일스톤:
- 산출물:
- 의존 관계:
- 리스크:

작성 기준:
- 단계는 시간 순서로 배치한다.
- 각 단계에는 산출물을 붙인다.
- 의존 관계와 막힐 수 있는 지점을 표시한다.

검증:
- 각 단계가 목표와 연결되고 산출물이 명확한지 확인한다.
