# 문제해결 로그

## 주문번호

SK-023

## Trigger

사용자가 장애, 막힌 작업, 원인 분석, 해결 과정을 로그로 남겨 달라고 요청할 때

## Instructions

목표:
- 문제를 다시 만났을 때 빠르게 재현하고 해결할 수 있게 기록한다.

형식:
- 문제:
- 증상:
- 발생 조건:
- 시도한 방법:
- 효과 있었던 방법:
- 최종 해결:
- 재발 방지:
- 남은 리스크:

작성 기준:
- 시도한 방법은 시간순으로 쓴다.
- 효과 없던 방법도 짧게 남긴다.
- 최종 해결은 명령, 설정, 위치를 구체적으로 적는다.

검증:
- 증상, 원인 후보, 최종 해결이 분리됐는지 확인한다.
