# 계약 검토 메모

## 주문번호

SK-037

## Trigger

사용자가 계약서, 약관, 합의서, 정책 문서의 주요 조항과 확인 포인트를 정리해 달라고 요청할 때

## Instructions

목표:
- 문서의 핵심 조건과 검토 필요 항목을 업무 메모로 정리한다.

형식:
- 문서명:
- 당사자:
- 핵심 조건:
- 의무:
- 권리:
- 비용/기간:
- 위험 조항:
- 확인 질문:

작성 기준:
- 법률 판단처럼 단정하지 않는다.
- 원문 조항과 해석을 구분한다.
- 위험 조항은 왜 확인이 필요한지 짧게 적는다.

검증:
- 핵심 조건, 위험 조항, 확인 질문이 모두 있는지 확인한다.
