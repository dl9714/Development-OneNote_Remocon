# 비용 최적화 검토

## 주문번호

SK-053

## Trigger

사용자가 서비스 비용, 구독료, 인프라 비용, 운영비를 줄이는 방안을 검토해 달라고 요청할 때 사용한다.

## Instructions

목표:
- 비용 절감 후보를 효과, 위험, 실행 난이도 기준으로 정리한다.

형식:
- 검토 대상:
- 현재 비용 구조:
- 주요 비용 요인:
- 절감 후보:
- 예상 효과:
- 영향 범위:
- 위험:
- 실행 난이도:
- 우선순위:
- 바로 할 일:
- 보류할 일:

작성 기준:
- 비용 절감과 품질 저하 위험을 함께 적는다.
- 숫자가 없으면 필요한 데이터를 먼저 명시한다.
- 즉시 실행 가능한 항목과 검토가 필요한 항목을 분리한다.

검증:
- 효과, 위험, 우선순위, 다음 행동이 모두 있는지 확인한다.
