# 원노트 섹션 그룹 생성

## 주문번호

SK-004

## Trigger

사용자가 전자필기장 또는 기존 섹션 그룹 아래에 새 섹션 그룹을 만들어달라고 할 때 사용한다.

## Instructions

1. 부모가 될 전자필기장 ID 또는 섹션 그룹 ID를 먼저 확보한다.
2. 캐시된 `section_group_id`가 대상 부모라면 바로 사용한다.
3. `OpenHierarchy("그룹명", parentId, ref newId, cftFolder)`를 호출한다.
4. `GetHierarchy(parentId, hsSections, ref xml)`로 새 그룹 이름을 확인한다.
5. 필요하면 새 그룹 ID를 `docs/codex/onenote-targets.json`에 저장한다.

## PowerShell Pattern

```powershell
[void][System.Reflection.Assembly]::LoadWithPartialName("Microsoft.Office.Interop.OneNote")
$one = New-Object -ComObject OneNote.Application

$parentId = "{2716C2CA-1EA5-4697-9AE7-97380372C026}{1}{B0}"
$groupName = "새 섹션 그룹"
$newGroupId = ""

$one.OpenHierarchy(
    $groupName,
    $parentId,
    [ref]$newGroupId,
    [Microsoft.Office.Interop.OneNote.CreateFileType]::cftFolder
)

$one.NavigateTo($newGroupId)
```

## OneNote COM Guardrails

- 섹션 그룹은 섹션 안에 만들 수 없다. 부모는 전자필기장 또는 섹션 그룹이어야 한다.
- 새 그룹을 이후에도 자주 쓸 경우 대상 ID 캐시에 저장한다.
- 검증은 `GetHierarchy` 기준으로 한다.
