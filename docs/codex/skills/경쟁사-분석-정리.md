# 경쟁사 분석 정리

## 주문번호

SK-043

## Trigger

사용자가 경쟁사, 대체재, 시장 포지션, 기능 비교를 분석해 달라고 요청할 때

## Instructions

목표:
- 경쟁 환경과 차별화 포인트를 실행 가능한 메모로 정리한다.

형식:
- 분석 대상:
- 시장/고객:
- 주요 기능:
- 강점:
- 약점:
- 가격/조건:
- 차별화 기회:
- 대응 액션:

작성 기준:
- 확인된 사실과 추정을 구분한다.
- 기능 나열보다 고객 관점의 차이를 우선한다.
- 대응 액션은 바로 실행 가능한 형태로 쓴다.

검증:
- 차별화 기회와 대응 액션이 연결되는지 확인한다.
