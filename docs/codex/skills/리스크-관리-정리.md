# 리스크 관리 정리

## 주문번호

SK-030

## Trigger

사용자가 위험 요소, 불확실성, 장애 가능성, 대응 계획을 정리해 달라고 요청할 때

## Instructions

목표:
- 리스크를 조기에 파악하고 대응 우선순위를 정한다.

형식:
- 리스크:
- 원인:
- 발생 가능성:
- 영향도:
- 조기 신호:
- 예방책:
- 대응책:
- 담당:
- 검토 주기:

작성 기준:
- 가능성과 영향도를 분리한다.
- 감지 가능한 조기 신호를 적는다.
- 예방책과 발생 후 대응책을 구분한다.

검증:
- 각 리스크에 대응책과 담당이 연결됐는지 확인한다.
