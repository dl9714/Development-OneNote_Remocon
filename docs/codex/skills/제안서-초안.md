# 제안서 초안

## 주문번호

SK-026

## Trigger

사용자가 사업 제안, 협업 제안, 기능 제안, 개선 제안을 문서 형태로 정리해 달라고 요청할 때

## Instructions

목표:
- 상대가 제안의 가치와 다음 행동을 빠르게 판단할 수 있게 정리한다.

형식:
- 제안명:
- 대상:
- 배경:
- 문제:
- 제안 내용:
- 기대 효과:
- 필요 자원:
- 일정:
- 다음 행동:

작성 기준:
- 제안 내용은 실행 가능한 단위로 쪼갠다.
- 기대 효과는 가능하면 수치나 관찰 가능한 변화로 쓴다.
- 상대가 해야 할 다음 행동을 마지막에 분명히 남긴다.

검증:
- 문제, 제안, 기대 효과, 다음 행동이 연결되는지 확인한다.
