# 회고 액션플랜

## 주문번호

SK-048

## Trigger

사용자가 회고 내용을 다음 실행 계획, 개선 과제, 담당자 중심으로 바꿔 달라고 요청할 때 사용한다.

## Instructions

목표:
- 회고에서 나온 내용을 감상으로 끝내지 않고 다음 행동으로 전환한다.

형식:
- 회고 주제:
- 잘된 점:
- 아쉬운 점:
- 반복 패턴:
- 근본 원인:
- 유지할 것:
- 바꿀 것:
- 중단할 것:
- 액션 플랜:
- 담당자:
- 마감일:
- 다음 점검일:

작성 기준:
- 액션 플랜은 동사로 시작한다.
- 담당자나 마감일이 없으면 "확정 필요"로 표시한다.
- 너무 큰 개선안은 첫 단계 작업으로 쪼갠다.

검증:
- 각 액션에 담당자, 마감일, 점검 기준이 있는지 확인한다.
