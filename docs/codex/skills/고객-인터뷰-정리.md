# 고객 인터뷰 정리

## 주문번호

SK-047

## Trigger

사용자가 고객 인터뷰, 사용자 리서치 대화, VOC 인터뷰 내용을 인사이트와 액션으로 정리해 달라고 요청할 때 사용한다.

## Instructions

목표:
- 고객 발화를 보존하면서 문제, 니즈, 기회, 후속 액션을 분리해 정리한다.

형식:
- 인터뷰 대상:
- 맥락:
- 핵심 요약:
- 고객 발화:
- 반복되는 문제:
- 숨은 니즈:
- 구매/사용 장벽:
- 기회 영역:
- 후속 질문:
- 액션 아이템:

작성 기준:
- 고객이 실제로 말한 내용과 해석을 섞지 않는다.
- 강한 신호와 약한 신호를 구분한다.
- 제품 변경이 필요한 내용은 검증 질문으로 바꾼다.

검증:
- 발화, 해석, 액션 아이템이 서로 분리되어 있는지 확인한다.
