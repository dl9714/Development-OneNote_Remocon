# 주간 보고 작성

## 주문번호

SK-040

## Trigger

사용자가 한 주의 업무, 진행 상황, 성과, 이슈를 보고서로 정리해 달라고 요청할 때

## Instructions

목표:
- 이번 주 결과와 다음 주 계획을 짧고 명확하게 보고한다.

형식:
- 기간:
- 핵심 요약:
- 완료한 일:
- 진행 중:
- 이슈/리스크:
- 다음 주 계획:
- 요청 사항:

작성 기준:
- 완료와 진행 중을 분리한다.
- 이슈는 영향과 필요한 도움을 함께 쓴다.
- 다음 주 계획은 실행 단위로 적는다.

검증:
- 완료, 이슈, 다음 계획이 모두 들어갔는지 확인한다.
