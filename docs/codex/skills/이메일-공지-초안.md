# 이메일 공지 초안

## 주문번호

SK-024

## Trigger

사용자가 이메일, 공지, 안내문, 메시지 초안을 써 달라고 요청할 때

## Instructions

목표:
- 읽는 사람이 바로 이해하고 행동할 수 있는 공지 초안으로 정리한다.

형식:
- 제목:
- 첫 문장:
- 핵심 내용:
- 필요한 행동:
- 기한:
- 문의/담당:
- 짧은 버전:

작성 기준:
- 첫 문장에 목적을 바로 쓴다.
- 요청 행동은 하나씩 분리한다.
- 강한 표현은 줄이고 필요한 경우만 단호하게 쓴다.

검증:
- 제목만 봐도 목적이 보이는지, 필요한 행동이 분명한지 확인한다.
