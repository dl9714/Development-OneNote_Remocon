# 원노트 전자필기장 생성

## 주문번호

SK-005

## Trigger

사용자가 새 OneNote 전자필기장을 만들거나 OneDrive/로컬 경로의 전자필기장을 열어달라고 할 때 사용한다.

## Instructions

1. 새 전자필기장을 만들 파일 시스템 경로를 먼저 확정한다.
2. OneDrive 동기화 폴더를 쓰는 경우 실제 로컬 경로가 존재하는지 확인한다.
3. `OpenHierarchy(notebookPath, "", ref newId, cftNotebook)`를 호출한다.
4. `GetHierarchy("", hsNotebooks, ref xml)` 또는 `hsSections`로 열린 전자필기장을 검증한다.
5. 생성/열기 완료 후 자주 쓸 대상이면 `docs/codex/onenote-targets.json`에 추가한다.

## PowerShell Pattern

```powershell
[void][System.Reflection.Assembly]::LoadWithPartialName("Microsoft.Office.Interop.OneNote")
$one = New-Object -ComObject OneNote.Application

$notebookPath = "C:\Path\To\새 전자필기장"
$newNotebookId = ""

$one.OpenHierarchy(
    $notebookPath,
    "",
    [ref]$newNotebookId,
    [Microsoft.Office.Interop.OneNote.CreateFileType]::cftNotebook
)

$one.NavigateTo($newNotebookId)
```

## OneNote COM Guardrails

- 웹 OneDrive URL이 아니라 OneNote가 열 수 있는 로컬/동기화 경로를 우선 사용한다.
- 경로가 애매하면 실행하지 말고 경로 확인부터 한다.
- 검증은 `GetHierarchy` 결과로 한다.
