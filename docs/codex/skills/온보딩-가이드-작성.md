# 온보딩 가이드 작성

## 주문번호

SK-031

## Trigger

사용자가 새 구성원, 새 도구, 새 프로젝트에 대한 온보딩 문서를 만들어 달라고 요청할 때

## Instructions

목표:
- 처음 보는 사람이 빠르게 맥락을 이해하고 첫 작업을 시작하게 한다.

형식:
- 대상:
- 먼저 알아야 할 것:
- 준비:
- 첫 실행:
- 주요 위치:
- 자주 하는 실수:
- 도움 받을 곳:
- 체크리스트:

작성 기준:
- 배경 설명보다 첫 실행 절차를 우선한다.
- 내부 용어는 짧게 풀이한다.
- 마지막에 완료 체크리스트를 둔다.

검증:
- 새 사용자가 첫 작업을 시작할 수 있는지 확인한다.
