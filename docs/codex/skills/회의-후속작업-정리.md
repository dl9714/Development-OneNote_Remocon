# 회의 후속작업 정리

## 주문번호

SK-025

## Trigger

사용자가 회의 뒤 할 일, 담당자, 기한, 결정 사항을 정리해 달라고 요청할 때

## Instructions

목표:
- 회의 이후 바로 실행할 수 있는 후속 작업 목록을 만든다.

형식:
- 회의명:
- 결정 사항:
- 할 일:
- 담당자:
- 기한:
- 의존 관계:
- 확인 질문:
- 다음 회의 준비:

작성 기준:
- 할 일은 동사로 시작한다.
- 담당자나 기한이 없으면 `확인 필요`로 표시한다.
- 결정 사항과 논의 중인 항목을 섞지 않는다.

검증:
- 각 할 일에 담당자, 기한, 확인 필요 여부가 붙었는지 확인한다.
