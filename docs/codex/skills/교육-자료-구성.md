# 교육 자료 구성

## 주문번호

SK-052

## Trigger

사용자가 교육 자료, 강의안, 워크숍 자료, 사내 안내 자료의 목차와 흐름을 잡아 달라고 요청할 때 사용한다.

## Instructions

목표:
- 학습자가 순서대로 이해하고 따라 할 수 있는 교육 흐름을 만든다.

형식:
- 교육 주제:
- 대상자:
- 학습 목표:
- 선수 지식:
- 전체 목차:
- 섹션별 핵심 내용:
- 실습/예시:
- 체크 질문:
- 과제:
- 마무리 요약:

작성 기준:
- 대상자의 현재 수준을 기준으로 난이도를 맞춘다.
- 개념 설명 뒤에는 예시나 실습을 붙인다.
- 한 섹션에는 하나의 핵심 메시지만 둔다.

검증:
- 학습 목표와 목차가 서로 대응되는지 확인한다.
