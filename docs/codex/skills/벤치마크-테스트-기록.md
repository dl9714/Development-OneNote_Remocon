# 벤치마크 테스트 기록

## 주문번호

SK-049

## Trigger

사용자가 성능 비교, 속도 테스트, 모델/도구/기능 벤치마크 결과를 기록하거나 비교표로 정리해 달라고 요청할 때 사용한다.

## Instructions

목표:
- 테스트 조건과 결과를 분리해 나중에 재현 가능한 기록으로 만든다.

형식:
- 테스트 목적:
- 비교 대상:
- 테스트 환경:
- 입력 조건:
- 측정 지표:
- 결과 요약:
- 상세 결과:
- 관찰:
- 예외/오차:
- 결론:
- 추천 선택:
- 재테스트 조건:

작성 기준:
- 숫자, 버전, 설정값을 가능한 한 보존한다.
- 빠른 것과 안정적인 것을 구분한다.
- 결과가 부족하면 결론을 보류하고 필요한 추가 실험을 적는다.

검증:
- 테스트 환경, 지표, 결과, 결론이 모두 있는지 확인한다.
