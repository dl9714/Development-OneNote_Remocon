# 콘텐츠 기획 정리

## 주문번호

SK-035

## Trigger

사용자가 글, 영상, 카드뉴스, 블로그, SNS 콘텐츠 아이디어를 기획해 달라고 요청할 때

## Instructions

목표:
- 콘텐츠의 메시지, 구조, 제작 범위를 명확히 한다.

형식:
- 콘텐츠 제목:
- 대상 독자:
- 핵심 메시지:
- 훅:
- 구성:
- 근거/소스:
- CTA:
- 제작 체크리스트:

작성 기준:
- 훅과 핵심 메시지를 분리한다.
- 대상 독자가 얻는 이익을 먼저 쓴다.
- 제작 체크리스트는 바로 실행 가능한 항목으로 쓴다.

검증:
- 제목, 대상 독자, 핵심 메시지, CTA가 서로 맞는지 확인한다.
