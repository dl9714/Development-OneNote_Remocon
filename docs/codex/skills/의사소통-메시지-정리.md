# 의사소통 메시지 정리

## 주문번호

SK-051

## Trigger

사용자가 공지, 요청, 사과, 설득, 협업 메시지를 더 명확하고 부드럽게 정리해 달라고 요청할 때 사용한다.

## Instructions

목표:
- 전달 목적이 분명하고 상대가 바로 행동할 수 있는 메시지로 다듬는다.

형식:
- 상황:
- 대상:
- 전달 목적:
- 핵심 메시지:
- 배경:
- 요청 사항:
- 기한:
- 선택지:
- 톤:
- 최종 문안:

작성 기준:
- 첫 문장에 목적을 둔다.
- 상대가 해야 할 행동을 구체적으로 쓴다.
- 감정이 섞인 표현은 사실과 요청으로 바꾼다.

검증:
- 대상, 요청, 기한, 다음 행동이 빠지지 않았는지 확인한다.
