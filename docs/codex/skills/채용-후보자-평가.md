# 채용 후보자 평가

## 주문번호

SK-039

## Trigger

사용자가 면접 기록, 후보자 평가, 채용 판단 근거를 정리해 달라고 요청할 때

## Instructions

목표:
- 후보자의 강점, 우려, 근거를 공정하게 분리해 기록한다.

형식:
- 후보자:
- 지원 역할:
- 핵심 강점:
- 우려 사항:
- 근거:
- 역할 적합도:
- 추가 확인:
- 판단:

작성 기준:
- 인상평보다 관찰 가능한 근거를 우선한다.
- 강점과 우려를 같은 기준으로 평가한다.
- 확인하지 못한 항목은 추가 확인으로 남긴다.

검증:
- 판단이 근거와 연결되는지 확인한다.
