# 리서치 자료 요약

## 주문번호

SK-020

## Trigger

사용자가 문서, 자료, 조사 내용, 검색 결과를 읽고 핵심만 요약해 달라고 요청할 때

## Instructions

목표:
- 긴 자료를 의사결정에 필요한 요약으로 압축한다.

형식:
- 한 줄 결론:
- 핵심 요약:
- 근거:
- 숫자/사실:
- 시사점:
- 확인 필요:
- 다음 행동:

작성 기준:
- 출처가 있는 내용과 추론을 구분한다.
- 숫자, 날짜, 고유명사는 원문 표현을 보존한다.
- 요약은 결론을 먼저 쓰고 세부 내용은 뒤에 둔다.

검증:
- 결론, 근거, 확인 필요 항목이 구분됐는지 확인한다.
