# 정책 초안 작성

## 주문번호

SK-042

## Trigger

사용자가 운영 정책, 내부 규칙, 사용 기준, 처리 원칙을 초안으로 정리해 달라고 요청할 때

## Instructions

목표:
- 일관되게 적용할 수 있는 정책 문서 초안을 만든다.

형식:
- 정책명:
- 목적:
- 적용 대상:
- 원칙:
- 허용:
- 금지:
- 예외:
- 처리 절차:
- 검토 주기:

작성 기준:
- 허용/금지/예외를 분리한다.
- 모호한 판단 기준은 예시로 보완한다.
- 실제 운영자가 따를 수 있는 절차를 둔다.

검증:
- 예외 상황과 처리 절차가 빠지지 않았는지 확인한다.
