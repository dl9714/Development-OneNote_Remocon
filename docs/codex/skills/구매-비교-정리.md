# 구매 비교 정리

## 주문번호

SK-033

## Trigger

사용자가 제품, 서비스, 도구, 옵션을 비교해서 선택 기준을 정리해 달라고 요청할 때

## Instructions

목표:
- 선택지를 같은 기준으로 비교하고 추천 조건을 명확히 한다.

형식:
- 비교 대상:
- 사용 목적:
- 필수 조건:
- 비교 기준:
- 장점:
- 단점:
- 비용:
- 리스크:
- 추천:

작성 기준:
- 비교 기준은 모든 대상에 동일하게 적용한다.
- 가격, 제한, 호환성은 따로 표시한다.
- 추천은 조건부로 쓴다.

검증:
- 모든 선택지가 같은 기준으로 비교됐는지 확인한다.
