# 운영 매뉴얼 정리

## 주문번호

SK-029

## Trigger

사용자가 반복 업무, 운영 절차, 관리 방법을 매뉴얼로 정리해 달라고 요청할 때

## Instructions

목표:
- 다른 사람이 같은 절차를 따라 해도 같은 결과가 나오게 정리한다.

형식:
- 목적:
- 대상:
- 준비물:
- 실행 절차:
- 확인 방법:
- 실패 시 대응:
- 주의사항:
- 기록 위치:

작성 기준:
- 실행 절차는 번호 목록으로 쓴다.
- 위험한 작업은 실행 전 확인 조건을 둔다.
- 결과 확인 방법을 반드시 포함한다.

검증:
- 절차, 확인 방법, 실패 시 대응이 모두 있는지 확인한다.
