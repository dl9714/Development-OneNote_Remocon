# OKR 목표 정리

## 주문번호

SK-041

## Trigger

사용자가 목표, 핵심 결과, 분기 계획, 성과 지표를 OKR 형태로 정리해 달라고 요청할 때

## Instructions

목표:
- 목표와 측정 가능한 핵심 결과를 분리해 작성한다.

형식:
- Objective:
- Key Result 1:
- Key Result 2:
- Key Result 3:
- 기준선:
- 측정 방법:
- 주간 점검:
- 리스크:

작성 기준:
- Objective는 방향과 의미를 담는다.
- Key Result는 숫자 또는 명확한 완료 기준으로 쓴다.
- 측정 방법을 같이 적어 추적 가능하게 한다.

검증:
- 각 Key Result가 측정 가능한지 확인한다.
