# User Skills Drafts

이 폴더는 앱의 `코덱스` 탭에서 만든 스킬 초안을 저장하는 공간이다.

OneNote COM 호출 방식, 대상 ID 우선순위, 작업별 PowerShell 패턴, 완료 검증 기준은 사용자 스킬이 아니라 `docs/codex/instructions`의 코덱스 전용 지침에서 관리한다.

## 저장 형식

각 스킬은 Markdown 파일 하나로 저장한다.

- `# 스킬 이름`
- `## 주문번호`
- `## Trigger`
- `## Instructions`

주문번호는 `SK-001` 형식을 기본으로 쓴다. 사용자가 "SK-001 조회해서 그 방식대로 해"라고 지시하면, 먼저 `skill-order-index.md` 또는 이 폴더의 Markdown 파일에서 해당 주문번호를 찾고 그 스킬 지시문을 따른다.

## 운영 기준

- 스킬은 반복되는 사용자 의도, 글쓰기 형식, 출력 기준을 담는다.
- OneNote 조작 최적화나 구현 절차는 사용자 스킬에 넣지 않는다.
- 검증 기준은 사용자가 확인할 결과 기준으로 적는다.

## 우선순위

- 나의 기본 글쓰기 형식
- 업무일지 정리
- 회의록 정리
- 아이디어 정리
- 작업 체크리스트
