# Codex Skills Drafts

이 폴더는 앱의 `코덱스` 탭에서 만든 스킬 초안을 저장하는 공간이다.

## 저장 형식

각 스킬은 Markdown 파일 하나로 저장한다.

- `# 스킬 이름`
- `## 주문번호`
- `## Trigger`
- `## Instructions`
- `## OneNote COM Guardrails`

주문번호는 `SK-001` 형식을 기본으로 쓴다. 사용자가 "SK-001 조회해서 그 방식대로 해"라고 지시하면, 먼저 `skill-order-index.md` 또는 이 폴더의 Markdown 파일에서 해당 주문번호를 찾고 그 스킬 지시문을 따른다.

## 운영 기준

- 스킬은 자연어 지시문과 실제 OneNote COM 작업 절차를 같이 담는다.
- 대상 OneNote 경로와 캐시된 ID가 있으면 명시한다.
- 완료 검증 방법은 `GetHierarchy` 또는 `GetPageContent` 기준으로 적는다.
- 화면 클릭 자동화는 마지막 fallback으로만 적는다.

## 우선순위

- 나의 기본 글쓰기 형식
- 임시 메모 페이지 추가
- 새 섹션 생성
- 새 섹션 그룹 생성
- 새 전자필기장 생성
