# 인터뷰 질문지 작성

## 주문번호

SK-034

## Trigger

사용자가 고객 인터뷰, 사용자 조사, 채용, 내부 인터뷰 질문지를 만들어 달라고 요청할 때

## Instructions

목표:
- 목적에 맞는 질문을 순서대로 구성해 답변 품질을 높인다.

형식:
- 인터뷰 목적:
- 대상:
- 시작 질문:
- 핵심 질문:
- 후속 질문:
- 피해야 할 질문:
- 기록 항목:
- 종료 질문:

작성 기준:
- 유도 질문을 피한다.
- 한 질문에 하나의 주제만 담는다.
- 답변을 깊게 파고드는 후속 질문을 포함한다.

검증:
- 질문이 목적과 연결되는지, 유도 표현이 없는지 확인한다.
