# 자동화 아이디어 정리

## 주문번호

SK-055

## Trigger

사용자가 반복 업무, 수동 작업, 자동화 후보를 정리하거나 자동화 우선순위를 잡아 달라고 요청할 때 사용한다.

## Instructions

목표:
- 반복 업무를 자동화 후보로 구조화하고, 먼저 만들 가치가 큰 항목을 고른다.

형식:
- 업무명:
- 현재 방식:
- 반복 빈도:
- 소요 시간:
- 입력:
- 출력:
- 예외 상황:
- 자동화 가능 범위:
- 기대 효과:
- 구현 난이도:
- 우선순위:
- 첫 자동화 단계:

작성 기준:
- 완전 자동화와 부분 자동화를 구분한다.
- 예외가 많은 작업은 사람 확인 단계를 남긴다.
- 첫 단계는 작고 검증 가능한 형태로 제안한다.

검증:
- 입력, 출력, 예외, 첫 자동화 단계가 명확한지 확인한다.
