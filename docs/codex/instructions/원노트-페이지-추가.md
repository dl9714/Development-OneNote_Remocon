# 원노트 페이지 추가

## 내부 용도

사용자가 OneNote 특정 섹션에 새 페이지를 만들고 제목/본문을 작성해 달라고 할 때 사용한다.

## 필요 값

- 대상 `Section ID`
- 페이지 제목
- 페이지 본문
- 사람이 읽는 대상 경로

## 절차

1. 캐시에서 대상 `section_id`를 먼저 확인한다.
2. 캐시가 없으면 `원노트-대상ID-찾기.md` 기준으로 섹션 ID를 찾는다.
3. 대상 `Section ID`로 `CreateNewPage`를 호출한다.
4. 새 페이지 XML을 `GetPageContent`로 읽는다.
5. XML 파서로 제목과 본문 outline을 수정한다.
6. `UpdatePageContent`로 반영한다.
7. 화면 확인 요청이 있을 때만 마지막에 `NavigateTo(pageId)`를 호출한다.

## 주의

- 페이지 추가만 할 때는 전체 `hsPages` 조회를 피한다.
- 제목/본문 반영은 문자열 치환보다 XML 노드 수정으로 처리한다.
- 본문 줄바꿈은 여러 `OE`로 나누어 넣는다.

## 검증

- `GetPageContent(pageId)`에서 제목과 본문 일부가 들어갔는지 확인한다.
- 실패하면 대상 경로, `Section ID`, Page ID, 제목을 함께 기록한다.
