# 원노트 섹션 그룹 생성

## 내부 용도

사용자가 전자필기장 또는 기존 섹션 그룹 아래에 새 섹션 그룹을 만들어 달라고 할 때 사용한다.

## 필요 값

- 부모 전자필기장 ID 또는 부모 `SectionGroup ID`
- 새 섹션 그룹 이름
- 대상 상위 경로

## 절차

1. 부모가 전자필기장인지 섹션 그룹인지 먼저 확정한다.
2. 캐시에 부모 ID가 있으면 먼저 사용한다.
3. 캐시가 없으면 `원노트-대상ID-찾기.md` 기준으로 부모 ID를 찾는다.
4. `OpenHierarchy("그룹명", parentId, ref newId, cftFolder)`를 호출한다.
5. 생성 직후 ID가 준비되지 않을 수 있으므로 필요하면 짧게 대기하고 다시 확인한다.

## 주의

- 섹션 안에는 섹션 그룹을 만들 수 없다.
- 같은 이름이 이미 있으면 생성 전 사용자 의도에 맞는지 확인한다.
- 화면 확인 요청이 없으면 `NavigateTo`는 필수가 아니다.

## 검증

- `GetHierarchy(parentId, hsSections, ref xml)`에서 새 섹션 그룹 이름과 ID를 확인한다.
- 반복 사용 대상이면 새 `section_group_id`를 캐시에 저장한다.
