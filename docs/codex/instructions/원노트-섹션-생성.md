# 원노트 섹션 생성

## 내부 용도

사용자가 특정 섹션 그룹 아래에 새 섹션을 만들어 달라고 할 때 사용한다.

## 필요 값

- 대상 `SectionGroup ID`
- 새 섹션 이름
- 대상 전자필기장 또는 상위 섹션 그룹 경로

## 절차

1. 캐시에서 대상 `section_group_id`를 먼저 확인한다.
2. 캐시가 없으면 `원노트-대상ID-찾기.md` 기준으로 상위 섹션 그룹 ID를 찾는다.
3. 새 섹션 이름에서 파일명으로 쓸 수 없는 문자를 정리한다.
4. 이름 끝에 `.one` 확장자를 붙인다.
5. `OpenHierarchy("섹션명.one", sectionGroupId, ref newId, cftSection)`를 호출한다.
6. 생성 직후 ID가 준비되지 않을 수 있으므로 필요하면 짧게 대기하고 다시 확인한다.

## 주의

- 섹션은 섹션 그룹이나 전자필기장 아래에 만든다.
- 같은 이름이 이미 있으면 생성 전 사용자 의도에 맞는지 확인한다.
- 화면 확인 요청이 없으면 `NavigateTo`는 필수가 아니다.

## 검증

- `GetHierarchy(sectionGroupId, hsSections, ref xml)`에서 새 섹션 이름과 ID를 확인한다.
- 반복 사용 대상이면 새 `section_id`를 캐시에 저장한다.
