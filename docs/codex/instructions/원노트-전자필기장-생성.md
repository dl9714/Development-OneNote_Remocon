# 원노트 전자필기장 생성

## 내부 용도

사용자가 새 OneNote 전자필기장을 만들거나 로컬/동기화 경로의 전자필기장을 열어 달라고 할 때 사용한다.

## 필요 값

- 전자필기장 이름
- OneNote가 열 수 있는 로컬 또는 동기화 폴더 경로

## 절차

1. 저장 위치가 실제 파일 시스템 경로인지 확인한다.
2. 웹 OneDrive URL만 있으면 그대로 쓰지 말고 동기화 폴더 또는 로컬 경로를 확인한다.
3. 대상 폴더가 없으면 생성 가능 여부를 확인한 뒤 만든다.
4. `OpenHierarchy(notebookPath, "", ref newId, cftNotebook)`를 호출한다.
5. 생성 직후 계층 반영이 늦을 수 있으므로 필요하면 짧게 대기하고 다시 확인한다.

## 주의

- 경로가 애매하면 전자필기장 생성보다 경로 확인을 먼저 한다.
- 같은 이름의 열린 전자필기장이 있으면 경로까지 확인한다.
- 화면 확인 요청이 없으면 `NavigateTo`는 필수가 아니다.

## 검증

- `GetHierarchy('', hsNotebooks, ref xml)` 또는 `GetHierarchy('', hsSections, ref xml)`에서 열린 전자필기장을 확인한다.
- 반복 사용 대상이면 전자필기장 ID와 기본 작업 섹션 ID를 캐시에 저장한다.
